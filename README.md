# app-jiuyouent-resource

本仓库用于归档和发布多个独立的 HTML 页面资源。页面内容涵盖不同主题，不针对特定域名或网站。

## 项目简介

`app-jiuyouent-resource` 是一个静态 HTML 页面集合仓库，主要用途为：
- 存储独立、自包含的 HTML 文件
- 提供简单的页面托管与归档功能
- 便于通过 GitHub Pages 或其他静态服务访问

仓库不涉及后端逻辑、数据库或动态内容，所有页面均为纯前端实现。

## 目录说明

```
├── index.html          # 仓库导航页（可选）
├── page1/              # 页面文件夹（示例）
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

- 每个页面通常位于独立文件夹内，便于管理
- 文件夹命名应反映页面主题或用途（无具体示例）
- 根目录下可放置全局资源（如样式、脚本、图片等）

## 页面归档说明

- 所有 HTML 页面均为静态文件，不依赖外部服务器
- 页面内资源（CSS、JS、图片）建议使用相对路径引用
- 仓库不保证所有页面长期可用，请以实际文件为准
- 页面内容可能随时间更新或移除

## 维护说明

- 本仓库由维护者不定时更新
- 欢迎通过 Issue 或 Pull Request 提交页面建议或修正
- 提交新页面时，请确保：
  - 文件结构清晰
  - 无外部依赖或已内联
  - 不包含恶意代码或敏感内容
- 仓库不承诺对提交内容进行审核或合并

## 使用方式

1. 克隆仓库到本地
2. 打开对应页面的 HTML 文件即可查看
3. 或通过 GitHub Pages 等服务部署后访问
